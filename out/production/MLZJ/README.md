# MLZJ
MLZJ Projektmunka
Egyetemi Bicikli flotta
Terv leírása: 
Egy flotta kezelése. Járművek nyomon követése és szervizintervallumának figyelése egy grafikus felületen keresztül.

Megvalósítandó részegységek:
•	Egy átlátható grafikus felület létrehozása
  oKülön rendszergazdai és felhasználói
  oKülön jogosultságok
•	A flotta összes járművének a nyomon követése és ellenőrzése
•	Szervizintervallum követése
•	Parkolók férőhelyének lekérdezése
•	Bérlők visszakövetése és eltárolása
•	Felhasználói fiók létrehozása és használata
•	Bérlés várható ára
•	Több fajta bicikli (elektromos/hagyományos)

Részletes leírás:
•	Felhasználói felület:
Telephelyek elhelyezkedése, és használatra kész járművek száma, tarifák kiszámítása.
•	Rendszergazdai felület:
A felhasználói jogokon felül, telephelyek létrehozása, flotta bővítése, biciklik állapotának lekérdezése, bérlők lekérdezése, biciklinkénti bevétel lekérdezése.
